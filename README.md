# maestra-core
 
# Description
Maestra (codename) is an immersive experience infrastructure platform. 
- Immersive experience - an experience that seeks to fully immerse the participants/viewers, often conveying strong emotions or visveral reactions.
- Infrastructure - the pieces necessary to build a functioning system, often hidden, but always foundational to the experience being created
- Platform - technology used to streamline creation, enabling more creativity and iteration

# Components
- Message/Event Bus - MQTT, OSC, AMQP, etc
- Peer-to-Peer Streaming (high-performance and reliability modes)
- Device/Fleet Management
- AI Asynchronous Workers
- Plugin Ecosystem - TouchDesigner, Max/MSP, Ableton Live
- Device Libraries - Arduino/ESP32/etc, Raspberry Pi
- Docker Configurations - straightforward setup locally or in the cloud
